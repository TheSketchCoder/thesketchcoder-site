# TheSketchCoder Labs

Anonymous technical portfolio for **TheSketchCoder Labs**.

> Independent experiments in AI-enabled frontend UX, system design, and browser graphics.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- MDX-ready writing support
- Static content modules

## Local Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates a production build.
- `npm run start` serves the production build.
- `npm run lint` runs ESLint.

## Privacy Boundaries

This site intentionally avoids identity-linked biographical details, analytics, authentication, databases, AI API calls, and WebGL/WebGPU runtime code in v0.

## v0 Scope

- Home
- Labs
- Workbench
- Writing
- System Design
- About

Future iterations can add real MDX posts, project detail pages, visual demos, and progressive WebGPU enhancements.
